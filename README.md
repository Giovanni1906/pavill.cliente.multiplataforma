## Local Secrets (Maps)

Create local files for API keys (do not commit real keys):

Android:
- Copy `android/local.properties.example` to `android/local.properties`
- Set `MAPS_API_KEY=YOUR_ANDROID_MAPS_KEY`

iOS:
- Copy `ios/Flutter/Secrets.xcconfig.example` to `ios/Flutter/Secrets.xcconfig`
- Set `MAPS_API_KEY=YOUR_IOS_MAPS_KEY`
